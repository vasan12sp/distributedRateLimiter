Understanding Rate Limits
  • Endpoint: The API path pattern (use * for wildcards, e.g., /api/*)
  • Method: HTTP method (GET, POST, PUT, DELETE, or * for all)
  • Limit: Maximum number of requests allowed
  • Window: Time period in seconds for the rate limit
